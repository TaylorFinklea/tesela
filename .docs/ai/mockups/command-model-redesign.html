2 · Leader menu

or press Ctrl , — opens the leader without leaving insert mode (chord = pending decision)
A wide multi-column which-key grid (emacs which-key / which-key.nvim style) — short, not tall. Press a bucket letter to descend (g / w / b / n / i / p / v / a / t / ,), Esc/Backspace to ascend. Every registry command lives in exactly one bucket — the discoverable, complete keyboard map.
Self-contained design mockup for the 2026-06-16 command-model redesign spec. No app code, no network. The slash before/after contrast demonstrates paring; the leader demonstrates that every command has a bucket home. Colors + chord-chip styling mirror the real ChordMenu.svelte / Prism warm-dark theme.
